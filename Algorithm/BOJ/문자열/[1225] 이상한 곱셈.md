## ğŸ™†â€â™€ï¸ ë‚´ í’€ì´ (python)

```python
import sys
input = sys.stdin.readline

a, b = input().split()
unique_a = list(set(a))
unique_b = list(set(b))

answer = 0
for x in unique_a:
  cur = 0
  for y in unique_b:
    cur += int(x)*int(y)*b.count(y)
  cur *= a.count(x)
  answer += cur

print(answer)
```
